## 2. Price Management System

### 💰 Commission Structure (Admin Controlled)
Admin can set all the following from the Admin Panel:

| Setting | Description |
|---|---|
| **Per Unit Commission %** | Admin sets what % of each sold unit goes to platform |
| **Commission by Category** | Different commission rates per product category |
| **Commission by Seller Tier** | Regular seller vs. Verified seller different rates |
| **Verified User Charge** | Admin sets a fixed charge for verified badge on sellers |
| **Minimum Commission Amount** | Set minimum flat commission regardless of % |

### 🚚 Dropshipping vs Regular Pricing

| Type | Description |
|---|---|
| **Regular Pricing** | Standard product price set by supplier |
| **Dropshipping Pricing** | Admin sets an additional % markup on top of supplier price for dropshippers |
| **Dropshipper Commission** | Admin adds X% to the price list shown to dropshippers |

### 📊 Price List Display
- Suppliers see their own price list
- Dropshippers see price list with added dropshipping margin
- Buyers see final retail price
- Admin sees all pricing layers simultaneously

### 🚢 Shipment Pricing
- Admin adds shipment cost per zone/region
- Admin sets shipment pricing tiers (standard, express, economy)
- Supplier can set custom shipping for their products
- Final price shown at checkout = Product Price + Shipment Price

### 📈 Reporting & Price Analytics
- Admin can view sales by price tier
- Commission earned per product / per seller report
- Dropshipping profit margin report
- Best/worst performing price points
- All reports configurable from Admin Panel

## 10. Dropshipping System

### How It Works
1. Supplier lists product on platform
2. Supplier opts product into dropshipping program
3. Admin sets dropshipping markup % from admin panel
4. Dropshipper browses dropship-eligible products
5. Dropshipper sells product at their set price (min = platform price + markup)
6. Customer places order through dropshipper's listing
7. Platform notifies supplier to ship directly to customer
8. Dropshipper earns the difference between their price and platform price

### Dropshipper Features
- Browse dropship-eligible product catalog
- Import products to their store with one click
- Set their own selling price (above platform minimum)
- See profit margin before choosing products
- Automated order routing to supplier
- Commission/profit tracking dashboard

### Admin Dropshipping Controls
- Set global dropshipping markup %
- Set per-category dropshipping markup
- Approve/reject dropshippers
- View all dropshipping transactions
- Turn dropshipping feature ON/OFF
