Getting Started

Four steps to make your first API call in under 5 minutes.

1
Create an Account
Register as a supplier or buyer on Globex Sky. Business verification may be required for full access.
2
Generate an API Key
Navigate to API → Keys, create a new key, and copy your secret token. Store it securely — it won't be shown again.
3
Make Your First Request
Use your key in the Authorization: Bearer header to call any endpoint below.
4
Set Up Webhooks
Register webhook URLs to receive real-time push events for orders, payments, and shipment updates.

Test all API endpoints without affecting real data. Use our pre-seeded test data and simulate order flows safely. Base URL: https://sandbox.api.globexsky.com/v2

Code Examples

Copy-ready snippets for the most common API operations.

Fetch Products – JavaScript (fetch API) 
// Fetch first page of products from Globex Sky API
const API_KEY = 'gsx_your_api_key_here';
const BASE_URL = 'https://api.globexsky.com/v2';

async function fetchProducts(page = 1, limit = 20) {
  const response = await fetch(
    `${BASE_URL}/products?page=${page}&limit=${limit}`,
    {
      method: 'GET',
      headers: {
        'Authorization': `Bearer ${API_KEY}`,
        'Content-Type': 'application/json',
        'Accept': 'application/json'
      }
    }
  );
  if (!response.ok) throw new Error(`HTTP ${response.status}`);
  const data = await response.json();
  console.log('Products:', data.items);
  return data;
}

// Create a new order
async function createOrder(payload) {
  const res = await fetch(`${BASE_URL}/orders`, {
    method: 'POST',
    headers: { 'Authorization': `Bearer ${API_KEY}`, 'Content-Type': 'application/json' },
    body: JSON.stringify(payload)
  });
  return res.json();
}

fetchProducts(1, 20).then(console.log).catch(console.error);
Fetch Products – Python (requests) 
import requests

API_KEY = "gsx_your_api_key_here"
BASE_URL = "https://api.globexsky.com/v2"

HEADERS = {
    "Authorization": f"Bearer {API_KEY}",
    "Content-Type": "application/json",
    "Accept": "application/json",
}

def fetch_products(page=1, limit=20):
    params = {"page": page, "limit": limit}
    response = requests.get(f"{BASE_URL}/products", headers=HEADERS, params=params)
    response.raise_for_status()
    return response.json()

def create_order(payload: dict):
    response = requests.post(f"{BASE_URL}/orders", headers=HEADERS, json=payload)
    response.raise_for_status()
    return response.json()

def track_shipment(shipment_id: str):
    response = requests.get(f"{BASE_URL}/shipments/{shipment_id}", headers=HEADERS)
    response.raise_for_status()
    return response.json()

# Example usage
if __name__ == "__main__":
    products = fetch_products(page=1, limit=10)
    print(f"Found {products['total']} products")
Fetch Products – PHP (cURL) 
<?php

$apiKey  = 'gsx_your_api_key_here';
$baseUrl = 'https://api.globexsky.com/v2';

function gsxRequest($method, $endpoint, $data = null) {
    global $apiKey, $baseUrl;
    $ch = curl_init($baseUrl . $endpoint);
    curl_setopt_array($ch, [
        CURLOPT_RETURNTRANSFER => true,
        CURLOPT_CUSTOMREQUEST  => $method,
        CURLOPT_HTTPHEADER     => [
            'Authorization: Bearer ' . $apiKey,
            'Content-Type: application/json',
            'Accept: application/json',
        ],
    ]);
    if ($data) curl_setopt($ch, CURLOPT_POSTFIELDS, json_encode($data));
    $response = curl_exec($ch);
    $code     = curl_getinfo($ch, CURLINFO_HTTP_CODE);
    curl_close($ch);
    if ($code >= 400) throw new Exception("API Error: {$code}");
    return json_decode($response, true);
}

// Fetch products
$products = gsxRequest('GET', '/products?page=1&limit=20');
echo "Total: " . $products['total'] . PHP_EOL;

// Create an order
$order = gsxRequest('POST', '/orders', [
    'supplier_id' => 'sup_123',
    'items'       => [['product_id' => 'prod_456', 'qty' => 100]],
    'currency'    => 'USD',
]);

Authentication Guide

All API requests must be authenticated using Bearer token authentication.

Bearer Token

Include your API key as a Bearer token in every request's Authorization header.

Authorization: Bearer gsx_your_api_key_here

Never expose your API key in client-side code.

Security Best Practices

  • Rotate keys regularly via the Keys dashboard
  • Use environment variables to store keys server-side
  • Restrict key permissions to only required scopes
  • Monitor key usage via the analytics dashboard
  • Revoke compromised keys immediately
