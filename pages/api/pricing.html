Frequently Asked Questions

Common questions about our API pricing.

What counts as an API call?
Each HTTP request to the API counts as one API call, regardless of the response size. Batched endpoints count as one call per batch request.
What happens when I exceed my monthly limit?
On the Free tier, requests are rejected with a 429 status once the monthly quota is reached. On Growth, you'll receive an email alert at 80% usage and can purchase additional call packs at $0.001 per call.
Is there a free trial for the Growth plan?
Yes — the Growth plan includes a 14-day free trial with full access to all features. No credit card required to start the trial.
Can I use the API for commercial projects?
Yes, all plans including Free allow commercial use subject to our Terms of Service. Enterprise plans can include custom licensing agreements for white-label or reseller use cases.
Do calls roll over to the next month?
No, unused API calls do not roll over. Quotas reset on the first day of each billing month.
